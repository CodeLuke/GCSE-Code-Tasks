
Problem:

    
The fibonacci numbers start 1, 1, 2, 3, 5, 8, 13, ...
    
The next number in each sequence is given by adding the last
    
two terms.

    
The function fib takes an integer n and should return the
    
nth fibonacci number


Tests:

    
>>> fib(1)
    
1
    
>>> fib(2)
    
1
    
>>> fib(5)
    
5
    
>>> fib(10)
    
55
